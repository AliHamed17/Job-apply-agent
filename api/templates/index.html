Personal WhatsApp Bridge

Use your own WhatsApp account — no Meta approval needed

Agent URL (bridge/.env)
http://localhost:8000
API Token (bridge/.env)
JOB_AGENT_TOKEN=<same as SECRET_KEY in .env>
  1. Make sure Node.js ≥ 18 is installed
  2. Run: cd bridge && npm install
  3. Edit bridge/.env — set JOB_AGENT_URL and JOB_AGENT_TOKEN
  4. Run: node whatsapp_bridge.js — a QR code appears
  5. Open WhatsApp on your phone → Linked Devices → Link a Device
  6. Scan the QR code — session is saved, no re-scan needed next time
  7. The bridge watches your groups automatically. The status dot above turns green.

Bridge Configuration

bridge/.env options

WATCH_ALL_GROUPS
true = watch every group (default)
WATCH_ARCHIVED_ONLY
true = only archived groups (recommended for job groups)
GROUP_KEYWORDS
jobs,hiring,careers,referally,mid,senior
JOB_URL_ONLY
false = forward all URLs (catches short links)
LOG_LEVEL
info | verbose | silent

WhatsApp Cloud API Setup

Official Meta Business API — requires app approval

Callback URL
/webhook/whatsapp
Verify Token
Set WHATSAPP_VERIFY_TOKEN in .env
App Secret
Set WHATSAPP_APP_SECRET in .env
API Token (send messages back)
Set WHATSAPP_API_TOKEN in .env
Phone Number ID
Set WHATSAPP_PHONE_NUMBER_ID in .env
  1. Go to Meta for Developers → create a WhatsApp app
  2. Under WhatsApp → Configuration, paste the Callback URL & Verify Token
  3. Subscribe to the messages webhook field
  4. Set WHATSAPP_API_TOKEN + WHATSAPP_PHONE_NUMBER_ID in .env to get approval buttons sent back
  5. Add your phone to ALLOWED_SENDERS in .env
